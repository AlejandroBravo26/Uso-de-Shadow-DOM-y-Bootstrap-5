# Uso-de-Shadow-DOM-y-Bootstrap-5

Este repositorio contiene un componente web personalizado `<calculadora-basica>` que implementa operaciones matemáticas fundamentales (suma, resta, multiplicación y división).

Construido con **Web Components** (utilizando **Shadow DOM** para encapsular su estructura y estilos) y **Bootstrap 5** para una interfaz moderna y responsiva. Incluye validaciones básicas, manejo de errores (como división por cero) y un historial de operaciones.
